# Card backs

Drop **5 photos** into this folder named exactly:

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`
- `5.jpg`

Each face-down card on the table picks one at random (stable per render, not per card — so it leaks zero information about anyone's hand). If a file is missing, the card just shows the default striped back instead.

Tip: roughly square or 3:4 portrait crops look best. They'll be `cover`-cropped to the card aspect (about 7:10).
